# E11 Oracle / Metered-Gain Baseline Bars (r1)

Setup: 4 held-out objects (letter_A, letter_L, stripe [= the task's 'stripe_target', the only stripe-family object in src.paper_experiments], ring), OU gain drift rho=0.001, s=0.1, random_uniform basis, N=2K=2048 oversampled (K=1024) so the static_reference treatment matches the ceiling diagnostic's 2K static result. 5 gain treatments: oracle (divide by true a_n), metered_noisy (divide by a_n*(1+0.01*eta), eta~N(0,1), a calibrated meter with 1% relative noise), blind_agc (mean_agc_gain with window W=32, chosen once via a bias-variance sweep over logspace_windows minimizing scale_aligned_gain_error, following Fig. 3's convention -- not retuned per object/seed), no_correction (raw drifted measurements), static_reference (noiseless a_n=1 measurements). CNR/PSNR (src/metrics.py, no post-processing) reported over 5 independent drift-trace seeds.
Oracle-vs-blind gap per object: letter_A: oracle-blind CNR=0.035 (oracle=3.735, blind_agc=3.699), oracle-blind PSNR=-0.04 dB; letter_L: oracle-blind CNR=0.137 (oracle=5.536, blind_agc=5.399), oracle-blind PSNR=-0.07 dB; ring: oracle-blind CNR=0.112 (oracle=3.118, blind_agc=3.006), oracle-blind PSNR=0.11 dB; stripe: oracle-blind CNR=0.147 (oracle=2.024, blind_agc=1.877), oracle-blind PSNR=0.42 dB.
Ordering sanity check: static_reference CNR >= oracle CNR >= blind_agc CNR >= no_correction CNR holds for 4/4 objects (oracle>=blind) and 4/4 objects (blind>=none), consistent with the accountability spectrum from full knowledge of the gain down to none.
